Quick info

.ec-card-*

Introduced in version 1.0.

5 pre styled "cards" available.

Works best with the <div> element.

If you want to have a easy to customize card, you can use .ec-card. However, if you want a box with a little more styling on it, you should use one of the following:
  1. .ec-card-1
      About:
    • Great for large content, however, the it takes up only 97% of the parent elements space (The main container here uses .ec-card-1).
  2. .ec-card-2
      About:
    • Uses just a little less than 50% of the parent element, except for when the browser width is too small.
  3. .ec-card-3
      About:
    • Uses just a little less than 1/3 of the parent element width, except for when the browser width is too small.
  4. .ec-card-full
      About:
    • Uses all of the parent element width.
    • The height is at minimum 100%.
  5. .ec-card-wide
      About:
    • Uses all of the parent element width.
    • The height is auto.
Of course, you can add more styling, for example: To add a background color, just use .ec-*, but replace "*" with one of the pre-selected colors (You can find the list of colors here). Here is a example, using blue as the background: 
<div class="ec-card-1 ec-blue">
    <p>
      Text :)
    </p>
</div>
<--The background is blue :)-->
