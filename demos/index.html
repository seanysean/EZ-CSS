Learn EZ CSS

In order to code in EZ CSS, you must have at least a basic knowledge of:

HTML5

CSS3

And obviously EZ-CSS (Which you will learn about here!).

First, how to add the CSS to your webpage.

You can copy/paste from the code from here:

CSS

Or add this line of code to the <head> of your code.

NOTE: If the code is not able to reach you site for whatever reason, EZ CSS will not work on your site. Also, since it is frequently updated, you should host your own version using the method above. But, if you insist ;) 
<link href="https://chesswithsean.neocities.org/projects/ez-css/ez.css" rel="stylesheet" type="text/css" media="all">

Next, make sure that your page will look great on all devices.

Add the following code to the <head> of your code.

<meta name="viewport" content="width=device-width; initial-scale=1.0;">

Your code should look something like:

          <!DOCTYPE html>
          <html>
            <head>
              <link href="Your local CSS file." rel="stylesheet" type="text/css" media="all">
              <meta name="viewport" content="width=device-width; initial-scale=1.0;">
            </head>
            <body>
              <!-- Put your own stuff here -->
            </body>
          <html>
